Image Guide For Choosing File Types.

File Extension When, Where, How? Image Examples
JPEG Images JPEG files should be used when an image has many different colors. For instance, an image of a sunset, or like this little cute bird.
PNG Images PNG files should be used when an image has fewer colors or a large area of the same color. For instance, a logo or simple illustration.
GIF Images GIF files should be used when an image has fewer colors or a large area of the same color. For instance, a logo or simple illustration.
